# Contributing to WatchToku

Thank you for your interest in contributing to WatchToku! We welcome contributions from everyone. Here's how you can help:

## Getting Started

1. **Fork the repository** - Click the fork button on the GitHub page
2. **Clone your fork** - `git clone https://github.com/YOUR_USERNAME/WatchToku.git`
3. **Create a branch** - `git checkout -b feature/your-feature-name`
4. **Make your changes** - Edit files and add new features
5. **Commit your changes** - `git commit -m "Add your commit message"`
6. **Push to your fork** - `git push origin feature/your-feature-name`
7. **Submit a pull request** - Open a PR to the main repository

## Code Guidelines

- Write clear, descriptive commit messages
- Keep commits focused on a single feature or fix
- Test your changes before submitting
- Follow the existing code style

## Reporting Issues

If you find a bug or have a suggestion, please open an issue with:
- A clear title and description
- Steps to reproduce (if applicable)
- Expected vs actual behavior

## Community

Be respectful and constructive in all interactions. We follow the Contributor Covenant Code of Conduct.

Thank you for contributing!
